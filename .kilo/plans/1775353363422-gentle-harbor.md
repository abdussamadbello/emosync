# EmoSync Typography & Design Improvement Plan

## Overview

Replace Geist Sans with **Plus Jakarta Sans**, increase text sizes across the app, improve typographic hierarchy, and refine the design for better readability and a more calming, therapeutic feel — while keeping the neutral/monochrome color palette.

---

## Files to Modify

| File | Changes |
|------|---------|
| `frontend/app/layout.tsx` | Swap font import from Geist to Plus Jakarta Sans |
| `frontend/app/globals.css` | Update design tokens, add typography scale, improve base styles |
| `frontend/components/chat_view.tsx` | Redesign message bubbles, welcome screen, input area |
| `frontend/components/sidebar.tsx` | Improve nav item sizing, spacing, and typography |
| `frontend/components/ui/button.tsx` | Update button text sizes and padding |
| `frontend/app/dashboard/page.tsx` | Improve card typography and spacing |
| `frontend/app/journal/page.tsx` | Improve entry card typography |

## Step 1: Font Swap — Plus Jakarta Sans

### `frontend/app/layout.tsx`

**Current:**
```tsx
import { Geist, Geist_Mono } from "next/font/google";
const geistSans = Geist({ variable: "--font-geist-sans", subsets: ["latin"] });
const geistMono = Geist_Mono({ variable: "--font-geist-mono", subsets: ["latin"] });
```

**Change to:**
```tsx
import { Plus_Jakarta_Sans } from "next/font/google";
const plusJakarta = Plus_Jakarta_Sans({
  variable: "--font-sans",
  subsets: ["latin"],
  weight: ["300", "400", "500", "600", "700"],
});
```

- Remove Geist_Mono import (no longer needed)
- Keep the `className` template string updated to use the new variable
- Plus Jakarta Sans has built-in weights 300–800, giving us more typographic range

## Step 2: globals.css — Design Token Overhaul

### Typography Scale (add to `@theme inline`)

```css
@theme inline {
  /* Existing tokens kept as-is */
  
  /* New typography tokens */
  --text-xs: 0.75rem;       /* 12px — labels, metadata */
  --text-sm: 0.875rem;      /* 14px — body, inputs, nav items */
  --text-base: 1rem;        /* 16px — default body text */
  --text-lg: 1.125rem;      /* 18px — section headings */
  --text-xl: 1.25rem;       /* 20px — page headings */
  --text-2xl: 1.5rem;       /* 24px — major headings */
  --text-3xl: 1.875rem;     /* 30px — hero elements */
  --text-4xl: 2.25rem;      /* 36px — dashboard hero numbers */
}
```

### Base Typography Improvements

Add to `@layer base`:

```css
@layer base {
  * {
    @apply border-border outline-ring/50;
  }
  body {
    @apply bg-background text-foreground;
    font-size: var(--text-base);
    line-height: 1.6;
    letter-spacing: 0.01em;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
  }
  html {
    @apply font-sans;
  }
  h1, h2, h3, h4, h5, h6 {
    letter-spacing: -0.025em;
    line-height: 1.2;
  }
  p {
    line-height: 1.7;
  }
}
```

## Step 3: Chat View Redesign

### Message Bubbles

**Current:**
```tsx
className={`max-w-[80%] rounded-2xl px-4 py-2.5 text-sm leading-relaxed ...`}
```

**Change to:**
```tsx
className={`max-w-[85%] rounded-2xl px-5 py-3 text-[0.9375rem] leading-[1.7] ...`}
```

- Increase from `text-sm` (14px) to `15px` for better readability
- Increase padding from `px-4 py-2.5` to `px-5 py-3`
- Increase max-width from `80%` to `85%`
- Line height from `relaxed` (1.625) to `1.7`
- Add subtle shadow to assistant bubbles: `shadow-sm`
- User bubbles: keep solid primary bg, add `shadow-sm shadow-primary/5`

### Welcome Screen

**Current:**
```tsx
<h2 className="text-xl font-semibold tracking-tight">Welcome to EmoSync</h2>
<p className="mt-1.5 text-sm text-muted-foreground">Your safe space for emotional wellness</p>
```

**Change to:**
```tsx
<h2 className="text-2xl font-semibold tracking-tight text-foreground">Welcome to EmoSync</h2>
<p className="mt-2 text-base text-muted-foreground">Your safe space for emotional wellness</p>
```

- Welcome heading: `text-xl` → `text-2xl`
- Subtitle: `text-sm` → `text-base`
- Increase top margin on subtitle

### Input Area

**Current:**
```tsx
className="flex-1 bg-transparent py-1.5 text-sm outline-none placeholder:text-muted-foreground ..."
```

**Change to:**
```tsx
className="flex-1 bg-transparent py-2 text-[0.9375rem] outline-none placeholder:text-muted-foreground/60 ..."
```

- Increase input text size to 15px
- Increase padding
- Soften placeholder color

### Disclaimer Text

**Current:**
```tsx
className="mt-2 text-center text-[11px] text-muted-foreground/70"
```

**Change to:**
```tsx
className="mt-2.5 text-center text-xs text-muted-foreground/60"
```

- Bump from 11px to 12px (`text-xs`) for accessibility

## Summary of Typography Scale Changes

| Element | Before | After |
|---------|--------|-------|
| Body text | 14px (`text-sm`) | 15px (`text-[0.9375rem]`) |
| Input text | 14px | 15px |
| Nav items | 14px | 15px |
| Button text | 14px | 15px |
| Page headings | 20px (`text-xl`) | 24px (`text-2xl`) |
| Section headings | 18px (`text-lg`) | 20px (`text-xl`) |
| Welcome heading | 20px | 24px |
| Welcome subtitle | 14px | 16px |
| Disclaimer | 11px | 12px |
| Mood score | 36px (`text-4xl`) | 48px (`text-5xl`) |
| Journal content | 12px (`text-xs`) | 14px (`text-sm`) |
| Border radius | 10px | 12px |

---

## Design Principles Applied

1. **Readability first**: 15px base text is the sweet spot for screen reading
2. **Clear hierarchy**: Larger headings, distinct metadata sizing
3. **Breathing room**: Increased padding and spacing throughout
4. **Softer tones**: Subtle blue-gray tint in neutral palette for warmth
5. **Rounded comfort**: Increased border radius for a friendlier feel
6. **Accessibility**: All text meets WCAG AA contrast ratios
7. **Consistency**: Unified 15px base across all interactive and body text
